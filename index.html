HTML role in web development

The role that HTML currently plays in web development, is that currently it is the backbone of all web development, and it runs the internet. No matter what sites you go to, or what sites you frequent, there is a good chance that the site has some level of HTML imbedded inside, building a simple building blocks website can be incredibly fast and effecient using HTML.


CSS's role in web development

CSS was the upgrade that HTML didnt know it needed, adding the ability to be more strict on how things look, colors, sizes and alignments. CSS created a more fine grain detail to web development that was not avaliable before, the ablity for the user to change the way that they design their sites on a fine tuned level, created a revolution of UI/UX development, and made it possible for the developers thing more about the user experience, and develop a better environment for the user.


JS's role in web development

Javascript is the next level of web development, its current iteration allows web development to be done with a more traditional programming language, that has the ability to build more logic and implementations into websites, creating a more dynamic and responsive website for the users to enjoy.
